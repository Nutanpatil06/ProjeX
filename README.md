# 🚀 ProjeX – Smart Project Management System

A modern, full-stack **project management SaaS platform** built with **Next.js, Supabase, and TypeScript**, designed to help teams efficiently plan, track, and deliver projects using a powerful Kanban workflow system.

---

## ✨ Key Highlights

- ⚡ Real-time Kanban board with drag & drop task management
- 🔐 Secure authentication (Email, Google, GitHub via Supabase Auth)
- 👥 Team collaboration with role-based access control
- 📊 Project analytics and progress tracking
- 🧠 Smart task organization with priorities, due dates, and status tracking
- 📱 Fully responsive UI (desktop + mobile)
- 🌙 Dark mode support for better user experience

---

## 🧩 Core Features

### 📌 Kanban Workflow
- Drag & drop task movement between stages
- Customizable columns (To Do, In Progress, Done, etc.)
- Task filtering and search functionality

### 🔐 Authentication & Security
- Supabase authentication system
- OAuth login (Google & GitHub)
- Protected routes and session handling

### 📁 Project Management
- Create and manage multiple projects
- Assign tasks to team members
- Track deadlines and priorities
- Monitor project progress in real-time

### 👤 User System
- Personal user profiles
- Activity tracking (task updates, comments, changes)
- Project participation history

---

## 🛠 Tech Stack

- **Frontend:** Next.js, React, TypeScript
- **Styling:** Tailwind CSS
- **State Management:** Zustand / React Query
- **Backend:** Supabase (Auth, Database, Storage)
- **Database:** PostgreSQL (via Supabase)
- **Deployment:** Vercel (recommended)

## ⚙️ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/projex.git
cd projex
```
### 2. Install dependencies
```bash
npm install
```
### 3. Setup environment variables
Create .env.local file:
```bash
RESEND_API_KEY=your_key_here

NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
NEXT_PUBLIC_SUPABASE_STORAGE_URL=your_storage_url
```
### 4. Run development server
```bash
npm run dev
```
App will run at:
```bash
http://localhost:3000
```
